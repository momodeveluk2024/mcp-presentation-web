# MCP + CMS Presentation

An interactive web presentation about how MCP (Model Context Protocol) controls Content Management Systems.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deploy to Netlify

### Drag & Drop (Fastest)
1. Run `npm run build`
2. Go to https://app.netlify.com/drop
3. Drag the `dist` folder there
4. Done!

### From GitHub
1. Push to GitHub
2. Connect to Netlify
3. Build command: `npm run build`
4. Publish directory: `dist`

## Slides Overview

| # | Slide | Content |
|---|-------|---------|
| 1 | **Title** | Team names, presentation title, animated flow diagram |
| 2 | **What is MCP?** | Data flow animation with moving packets |
| 3 | **The Problem** | Rotating showcase of 5 pain points |
| 4 | **The Solution** | AI chat conversation with typing effect |
| 5 | **Create Post** | Live CMS dashboard animation |
| 6 | **Update Post** | Real-time content updating |
| 7 | **Delete Post** | Safe deletion with backups |

| 10 | **Thank You** | Looping demo of AI → MCP → CMS flow |
